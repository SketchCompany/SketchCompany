Sketchy Games Launcher

Unser Launcher für eure Spiele

Was ist der Sketchy Games Launcher?

Der Sketchy Games Launcher bietet dir die Möglichkeit alle Games und Softwares von uns herunterzuladen, sie zu verwenden und zu verwalten. Ein Launcher ist eine Anwendung bzw. ein Programm das alle diese Aufgaben übernimmt, um dir die Nutzung zu vereinfachen.

Features des Launchers

Games und Softwares

Mit dem Launcher erhaltet ihr den vollen Zugriff auf alle Games und Softwares sowie andere spannende Features. Ihr könnt sie ganz einfach aus dem Store herunterladen.

Friends und Dateien

In der zukünftigen Version könnt ihr mit dem Launcher auch Freunde finden. Durch unser neues System wird es möglich Freunde hinzuzufügen und dann auch später mit ihnen zu spielen. Außerdem könnt ihr ganz einfach Dateien, wie Speicherstände oder Weltdaten miteinander austauschen.

Automatische Updates

Nicht nur eure Games und Softwares werden aktualisiert, sondern auch der Launcher! Nach einem kurzem Hinweis beim öffnen des Launchers wird im Hintergrund das Update heruntergeladen und der Launcher aktualisiert sich dann von selbst.

Wie funktioniert der Launcher

Electron und Chromium

Der Launcher wird auf der Chromium Engine gerendert und basiert auf dem Electron Framework. Dabei handelt es sich um eine Anwendung die mit HTML, CSS und JavaScript entwickelt wird.

Sketch Company API

Durch die Verbindung zur Sketch Company API hat der Launcher vollen und dynamischen Zugriff auf die aktuellsten Daten, besonders für den Store. Außerdem wird somit auch die Account Verwaltung in der Anwednung möglich.

Sicherheit

Alle Daten werden mit einer sehr schwer lösbaren Verschlüsselung gesichert. Dabei achten wir darauf, dass alle wichtigen Daten mit einem komplexen Algorithmus sicher und versteckt bleiben.

Zukunft des Launchers

Friends System

Wie in jeden anderen Launchern soll auch im Sketchy Games Launcher ein System zum Zusammenspielen oder Arbeiten intigriet werden. So soll man im Launcher Freunde hinzufügen, mit ihnen chatten, vielleicht sogar einmal mit Voice callen und auf jeden fall auch zusammenspielen. Dazu kann man sich auch Dateien in der Größe von maximal 500MB zuschicken kann.

API Einbindung

Eine bessere Einbindung der API soll auch mit in den Launcher kommen und auch eine öffentliche API nicht nur für den Zugriff auf die Sketch Company Accounts sonder auch für kommende Features und Tools. Dabei kann durch ein API Token aus jeder Anwendung auf unsere REST API zugegriffen werden, die dann in JSON formatierte Daten zurückgibt.

Erweiterte Download Funktion

Auch die jetztige Download Funktion soll überarbeitet werden. Momentan wird immer das ganze neue Spiel bei einer neuen Version heruntergeladen, was bei der kleinen Größe der Spiele erst einmal kein großes Problem ist jedoch in der Zukunft bei größeren Projekten problematisch wird. Die neue Download Funktion setzt dabei auf manifestierungen und tracking jeder Dateien.

Payment Methods

Wir setzten unseren Fokus immer mehr auf hochwertigere Spiele und können dabei nicht immer alles aus eigener Tasche bezahlen. Deshalb werden in der Zukunft auch eigene Spiele für einen kleinen Preis erweblich gemacht. Natürlich wird es auch immer Rabatte oder Sonderaktionen geben in denen ihr unsere Spiele abstauben könnt. Unsere Spiele wollen wir außerdem auch in anderen Launchers anbieten, wie zum Beispiel Steam.
